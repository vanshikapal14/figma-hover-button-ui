# Figma Hover Button UI

This repository showcases a hover button UI concept designed in Figma.
The design focuses on micro-interactions, glow effects, and smooth visual
feedback suitable for modern mobile applications.

## Features
- Rounded pill-shaped button
- Hover / active visual state
- Glow and sparkle micro-interaction
- Dark theme friendly UI

## Tools Used
- Figma
